# TaskManager
Calendar2.0<br/>

## Функции
+ Регистрация/авторизация<br/>
+ Вывод календаря<br/>
+ Переход по месяца<br/>
+ Получение задач для текущего месяца и года<br/>
+ Создание, чтение, изменение, удаление задачи<br/>
+ Переход на страницу задачи<br/>
